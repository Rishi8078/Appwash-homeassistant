# AppWash Integration for Home Assistant

This integration allows you to monitor your AppWash washing machines and dryers in Home Assistant.

## Features
- Monitor washing machine availability and status
- Monitor dryer availability and status
- View account balance

## Installation
1. Install via HACS by adding this repository as a custom repository
2. Restart Home Assistant
3. Add the integration through the HA interface (Configuration -> Integrations -> Add Integration)
4. Enter your AppWash credentials

## Available Sensors
- Washing Machine Availability
- Dryer Availability
- Account Balance
